## 1. 基礎編：Vue.jsの基本操作

### 1-1. カウンター機能の実装
**課題意図**:
Vue.js の基本である「リアクティブなデータ（状態）」と「イベントハンドリング」を理解する。

**回答コード例**:
```vue
<script setup>
import { ref } from 'vue'

const count = ref(0) // リアクティブな変数を定義

const increment = () => {
    count.value++ // 値を更新（.valueが必要）
}
</script>

<template>
    <p>カウント: {{ count }}</p>
    <button @click="increment">カウントアップ</button> <!-- イベントバインディング -->
</template>
```

**解説**:
- `ref()`: データが変わったら画面も自動で更新したい場合に使用します。
- `@click`: ボタンが押されたときに関数を実行します。

### 1-2. タイトル変更機能（双方向バインディング）
**課題意図**:
画面からの入力をデータに反映させ、データが変われば画面も変わる「双方向バインディング」を理解する。

**回答コード例**:
```vue
<script setup>
import { ref } from 'vue'

const title = ref('初期タイトル')
</script>

<template>
    <h1>{{ title }}</h1>
    <!-- v-modelでデータと入力欄を同期 -->
    <input type="text" v-model="title"> 
</template>
```

**解説**:
- `v-model`: フォームの入力値と変数を「紐付け」ます。入力すれば変数が変わり、変数が変われば入力欄の中身も変わります。

### 1-3. リスト表示機能（現在挑戦中）
**課題意図**:
配列データを扱い、`v-for` を使って一覧表示する。また配列への追加操作を行う。

**ヒント**:
- 配列の定義: `const items = ref(['A', 'B'])`
- ループ表示: `<li v-for="item in items">{{ item }}</li>`
- 追加: `items.value.push('C')`
